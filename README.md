# Masul Digital — Website

Single-page marketing site for **Masul Digital**, a local digital & AI services agency in South East London (SE18).

## Pages
All sections live on one scrollable page: Home, Services (Falcon AI, Data Desk), Pricing, About, Contact.  
Detail pages for each service are in `pages/` (Services, Falcon AI, Data Desk, Pricing).

## Structure
```
index.html         Main single-page site
assets/            Styles, favicon, brand SVGs
pages/             Per-service detail pages
```

## Deploy
Static site — no build step. Push to GitHub and connect to Vercel (Framework: Other).
